      :> System Help

      Glytchlings is a seeded procedural Glytchling generator. The same shape and color values always recreate the same result.

      [ Utility Menu ]

      Help Icon

      • Opens the System Help menu.

      Bookmark Icon

      • Opens the Specimen Log.
      • Add Current adds the current Glytchling shape, color, trait toggles, and mode settings to the log.
      • Select a saved Glytchling from the log to load it into the app.
      • Clearing browser storage or cache on this device may remove your saved collection.

      Copy Icon

      • Copies a shareable URL with the current shape and color seeds, traits, and mode settings.

      Download Icon

      • Saves the current rendered Glytchling view as a PNG.
      • Filenames use a filesystem-safe version of the Glytchling name, shape seed, and color seed. Unsupported characters are replaced with -.

      [ Control Panel ]

      Shape

      • Enter a shape seed and press Load to restore a specific Glytchling form.

      Color

      • Enter a color seed and press Load to restore a specific palette.

      Glytch

      • Spawn generates a new Glytchling with a random shape seed and color seed.
      • Color generates a new color palette with a random seed for current Glytchling shape.
      • Cºrru_pt randomizes Glytchling Traits toggles and some Mode settings. {Can produce extreme results!}
      • Restore returns the current Glytchling to a clean, readable baseline with all parts enabled and symmetrical. {Use when too Cºrru_pt!}

      Mode

      • Sym toggles all Glytchling part symmetry uniformly on or off.
      • CRT toggles Glytchling monochrome terminal render mode on or off.
      • Gap toggles Glytchling dome/core separation on or off.
      • Pulse toggles Glytchling idle animation on or off.
      • Pause Icon freezes the current pulse frame.

      [ Trait Panel ]

      • Displays current Glytchling Name, Shape Seed, and Color Seed.
      • Bookmark Icon adds the current Glytchling shape, color, trait toggles, and mode settings to the Specimen Log without opening the log.
      • Hover a trait to highlight it on the main Glytchling.
      • Click a trait to pin that highlight until you click it again.
      • On enables or disables the part.
      • Sym enables or disables the part symmetry.
      • Flip swaps the side used when a part is asymmetrical.
