      :> System Help

      Glytchlings is a seeded procedural Glytchling generator. The same shape and color values always recreate the same result.

      User Settings

      [ Spawn! Button ]

        Generates a completely new shape and color seeded Glytchling with default Inspector toggles.

      [ Glytchling Panel ]

        • Magnifying Glass Icon opens the Inspector Panel, see section below.
        • Name is the generated Glytchling's name or nickname.
        • Type displays combined motion energy family and cry family.
        • Cry shows the current cry waveform. Click it to play Glytchling's cry.
        • Book Icon opens the Specimen Log, see section below.
        • Copy Icon copies a shareable URL with the current shape and color seeds, and any Inspector customizations.
        • Download Icon saves the current rendered Glytchling as a PNG.
        • Bookmark Icon saves the current Glytchling shape, color, and Inspector customizations to the log.

        [ Specimen Log ]

        • Collection shows the amount of saved Glytchlings out of the 200 limit.
        • Last Action shows the last performed specimen log action.
        • A hidden alert can display if log is full or issues uploading a log file.
        • Click a saved Glytchling from the log to load it back into the app.
        • Trash Icon removes the selected saved Glytchling from the log.
        • Upload Log imports a saved specimen log JSON file and rebuilds thumbnails locally.
        • Download Log exports the current specimen log as a JSON file.
        • Clearing browser storage or cache on this device may remove your saved collection.

        [ Inspector Panel ]

        Shape Tab

        • Enter a shape seed and press Load to restore a specific Glytchling form.
        • Reshape rolls a new shape seed while keeping the current color seed and toggle state.
        • Cºrru_pt randomizes the Shape Inspector toggles without changing the current seeds.
        • Sym toggles all part symmetry on or off.
        • Anim toggles all part animation on or off.
        • Restore returns the current Glytchling to its base readable baseline with all parts enabled, symmetrical, and unflipped.

          • Per Part Controls

        • Hover a part to highlight it on the main Glytchling.
        • Click a part to pin that highlight until you click it again.
        • On enables or disables the part.
        • Sym enables or disables the part symmetry.
        • Flip swaps the side used when a part is asymmetrical.
        • Anim enables or disables animation for the part.

        Color Tab

        • Enter a color seed and press Load to restore a specific base palette.
        • Recolor rolls a fresh base color seed for the current Glytchling.
        • De$yn𝓬 assigns fully random per-part H, S, and L values outside the normal palette families.
        • Crt Toggles Glytchling monochrome terminal render mode on or off.
        • Invrt Inverts the current Glytchling colors.
        • Restore clears current color edits and returns to the current color seed's default palette.

          • Per Part Controls

        • Hover a part to highlight it on the main Glytchling.
        • Click a part to pin that highlight until you click it again.
        • Pinned part displays HSL sliders to manually change part color.

        Identity Tab

        • Enter a name and press Nickname to rename the Glytchling. Nicknames only persist when saved to the specimen log.
        • Restore returns the current Glytchling to it's generated seed name.
