# Changelog

## v1.1.0 - 2026-03-31

- Added deterministic Glytchling cries, clickable cry waveform playback, and combined `Type` readouts from motion energy plus cry family.
- Added live part animation for dome, node, bits, digits, and treads with per-part `Anim` toggles and seeded motion energy families.
- Refactored the Inspector into `Shape`, `Color`, and `Identity` tabs with in-panel controls, nickname editing, and canonical specimen trait labels.
- Added per-part H/S/L color editing, global `CRT` and `INVRT` color controls, fully random `De$yn𝓬` palettes, and share-link support for color overrides.
- Added fullscreen display mode in Help Menu.
- Simplified the main UI with a dedicated `SPAWN!` action, hideable Inspector, updated Glytchling panel layout, and persistent Help-based user settings for mute and Inspector startup behavior.
- Expanded specimen log safeguards with collection limits, oversized import rejection, stricter input validation, and name-length normalization across typing, saving, and import.

## v1.0.1 - 2026-03-23

- Added specimen log import/export with JSON validation and local thumbnail rebuilding.
- Added persistent specimen log last-action status messaging.
- Added small in-app build label and changelog modal to System Help menu.
- Refined specimen log and help menu UI wording and controls.
- Removed the in-log Save Specimen button in favor of the trait panel quick-save bookmark.

## v1.0.0 - 2026-03-21

- First public release of Glytchlings.
- Deterministic shape, color, and name generation.
- Stateful share links and PNG export.
- Local specimen log favorites.
